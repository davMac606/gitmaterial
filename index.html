O que é Git?

Git é um sistema de controle de versão distribuído gratuito e de código aberto projetado para lidar com tudo, desde projetos pequenos a muito grandes, com rapidez e eficiência. O Git é melhor aproveitado quando usado em conjunto com a plataforma Github

Comandos básicos do Git

Para usar o Git, deve-se utilizar o terminal. Para iniciá-lo, basta clicar com o botão direito do mouse em uma pasta vazia e selecionar a opção "Git Bash Here".
OBS: Existe a opção Git GUI, porém existem ações mais fáceis de realizar no terminal(Bash), além de, por ser um terminal Unix, permite utilizar comandos do Linux em computadores com sistema operacional Windows.

Lista de comandos:

  • git init: Ao executar o comando git init, transforma a pasta atual num repositório local do Git.

  • git add: O comando git add é utilizado para começar a transferência de arquivos para o repositório. Para adicionar todos os arquivos, basta utilizar o comando git add .
    Para adicionar apenas um arquivo, a sintaxe é a seguinte: git add filename.extension. Exemplo: git add index.html

  • git commit: O comando git commit prepara os arquivos adicionados para a transferência. Sintaxe: git commit -m "título do commit" -m "Descrição(não necessária)"
    Exemplo: git commit -m "Adicionando index.html" -m "Adicionando a página principal ao repositório."

  • git push: O comando git push é a última etapa da transferência, efetivamente transferindo os arquivos do repositório local para o repositório remoto.
    Este comando pode ter várias sintaxes, mas a mais comum é git push remote branch
    Exemplo: git push origin main

  • git status: O comando git status exibe exatamente o status do repositório, exibindo arquivos adicionados mas não adicionados a um commit, branch atual, diferença entre branch remoto e branch local, entre outros.
    Sua sintaxe é simples, sendo somente git status
